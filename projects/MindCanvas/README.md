# MindCanvas

MindCanvas é um quadro digital interativo que permite criar, editar e organizar conteúdo de forma visual e intuitiva.

## Funcionalidades

- **Desenho Livre**: Use a ferramenta de desenho para criar esboços e anotações
- **Texto**: Adicione blocos de texto editáveis em qualquer lugar do canvas
- **Formas**: Crie formas geométricas básicas
- **Setas**: Conecte elementos com setas
- **Imagens**: Importe e posicione imagens no canvas
- **Emojis**: Adicione emojis para tornar seu canvas mais expressivo
- **Tema Claro/Escuro**: Alternar entre temas claro e escuro
- **Salvamento Automático**: Seu trabalho é salvo automaticamente no navegador
- **Exportação**: Exporte seu canvas como imagem PNG

## Como Usar

1. **Seleção**: Clique no ícone de cursor para selecionar e mover elementos
2. **Desenho**: Use a ferramenta de caneta para desenhar livremente
3. **Texto**: Clique no ícone de texto para adicionar um bloco de texto
4. **Formas**: Use a ferramenta de formas para criar elementos geométricos
5. **Setas**: Clique e arraste para criar setas conectando elementos
6. **Imagens**: Importe imagens do seu computador
7. **Emojis**: Selecione emojis da biblioteca para adicionar ao canvas
8. **Borracha**: Apague partes do desenho
9. **Salvar**: Salve seu trabalho no navegador
10. **Exportar**: Exporte o canvas como imagem

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Canvas API
- LocalStorage API
- Font Awesome (ícones)
- Google Fonts (Inter)

## Compatibilidade

O MindCanvas é compatível com os seguintes navegadores:
- Google Chrome (recomendado)
- Mozilla Firefox
- Microsoft Edge
- Safari

## Instalação

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/mindcanvas.git
```

2. Navegue até o diretório do projeto:
```bash
cd mindcanvas
```

3. Abra o arquivo `index.html` em seu navegador
